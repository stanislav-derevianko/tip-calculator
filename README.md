# Tip Calculator (Bill Splitter)

A simple Python CLI application to calculate tips and split a bill between multiple people.

This project was created as a learning exercise to practice basic Python programming, including user input handling, loops, and conditional logic.

## Features

- Calculates tip amount based on a given percentage  
- Splits the total bill between multiple people  
- Validates user input  
- Allows repeating the calculation  
- Runs in the command line (CLI)

## How to run

Make sure you have Python installed (Python 3 recommended).

Clone the repository and run:

```bash
python tip_calculator.py
